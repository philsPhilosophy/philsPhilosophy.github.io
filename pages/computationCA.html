How the 2D CA squares N: We seed N horizontal and N vertical particles. Each step, horizontal particles move right; vertical move down. When both are present in a cell simultaneously, the cell deposits a permanent black mark. Because every horizontal meets every vertical exactly once, the total marks = N×N.

This uses purely local rules: each cell’s next state depends only on its own content and immediate neighbors.

Primes:
  • White = unknown
  • Black = composite
  • Green = prime (confirmed & frozen)
  • Purple dots = active local “walkers” carrying period p

How the 1D sieve CA works: Cells represent integers 2,3,4,… laid left→right. A walker with period p moves one cell per tick. Each step it decrements a local countdown; when it reaches 0 on a live (unknown) cell, it marks it composite and resets its countdown to p. When a walker reaches a still-unknown cell (i.e., it has not been marked composite by any smaller period), that cell becomes a prime emitter: it is frozen green and spawns a brand-new walker with period p = cell’s number. No global knowledge—just local interactions.
